什么是单例模式？

保证一个类仅有一个实例，并提供一个访问它的全局访问点。

JavaScript中的单例模式

上面的单例模式的实现，更多更多的是接近传统面向对象语言中的实现，但JavaScript其实是一门无类语言，也正因为如此，生搬单例模式的概念并无意义。在 JavaScript 中创建对象的方法非常简单，单例模式的核心是确保只有一个实例，并提供全局访问。
